Vad är Angular JS?

Defintion:

“AngularJS is a framework for client-side Model–View–Controller (MVC) and Model–View–ViewModel (MVVM) architectures. The framework adapts and extends traditional HTML to present dynamic content through two-way data-binding that allows for the automatic synchronization of models and views. As a result, AngularJS de-emphasizes explicit DOM manipulation with the goal of improving testability and performance."

Template

 HTML och den markup som används.

View

 Vad användaren ser.

Model

 Den data som visas för användaren i vyn och den som användaren interagerar med.

Controller

 Logiken bakom vyn.

Directive

 Utökad funktionalitet av HTML med anpassade attribut och element.

Performance problems

  • Problem: Too many watchers
  • Solution: One time data bindings
  • Problem: Too much logic in html. in the {{}}-expressions
  • Solution: Use controller-functions for calculations. Or even better. Factories!

Varför ska man välja Angular?

  • Utvecklat och underhålls av Google.
  • Stort community. Alla har haft alla problem.
  • Framtidssäkert.

Fördelar med Angular:

  • Two-Way Data Binding
  • All template-utveckling sker i ren HTML
  • Dependency Injection
  • Utökad HTML med directives

Nackdelar med Angular:

  • Kan ganska lätt bli "tungt" för browsern, (många $digest loopar som körs och uppdateringar av DOMen)
  • Lite knepigt med SEO
